Comparison between HTTP and HTTPS

Aspect HTTP HTTPS
Security Not secure; data transmitted in plain text Secure; data encrypted using SSL/TLS
Protocol Operates over a standard and unencrypted connection (port 80) Uses a secure and encrypted connection (port 443)
URL Scheme URL starts with "http://" URL starts with "https://"
Security Indicator No visual indication in the browser Browser displays a padlock icon or the word "Secure" in the address bar
Use Cases Suitable for non-sensitive data Essential for handling sensitive data
